Place project photos here.

The site currently uses placehold.co images referenced directly in the HTML,
e.g. https://placehold.co/600x400/1A1A1A/C9A84C

To use your own photos:
1. Drop image files into this folder (e.g. curtain-01.jpg).
2. In gallery.html, replace the placehold.co src / data-full URLs with:
   src="assets/images/curtain-01.jpg"
3. Recommended sizes: gallery ~1200px wide, hero/story ~1600px wide.
